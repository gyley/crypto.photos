DOCUMENTATION

The Crypto Photos project logo:

The Crypto Photos project website: http://crypto.photos

Our team noticed that one of the often-unmet needs of developers is the availability of an exhaustive library of cryptocurrency asset icons. For this reason, the ResoNova team has compiled an easy-to-implement archive of standardized high quality "crypto photos".

FORMATTING AND SYNTAX

All source photos are formatted as .PNG and are sized at exactly 300x300 px

All photos displayed on the project website are rendered at 150x150 px

Search by asset symbol's first letter

To view all asset icons beginning with a specific letter of the alphabet, simply navigate to the index of that letter by using the URL format below: 

http://crypto.photos/x

where "x" is the letter of the alphabet that you wish to search.

Search for assets with non-alphabetic first characters

To vew all asset icons which begin with non-letter characters, navigate to the following URL: 

http://crypto.photos/num

Direct logo icon linking

To link directly to a specific icon use the following format:

http://crypto.icon/x/xyz.png

where "x" is the first letter of the asset symbol, and "xyz" is the lowercase asset symbol. Be sure to add the file type (.png) onto the end.

Duplicate Tickers

In the event that there are duplicate ticker symbols, numbers starting from "-001" will be appended to the end of the icon symbol in the url.

For example:

http://crypto.photos/x/xyz-001.png

Numbers are implemented according to token creation date.
